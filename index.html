Justify-Content

The justify-content property aligns the flex items on the main axis, aligned by default to the left.

The space-between value divides the empty space and inserts it between the flex items.

justify-content: flex-startjustify-content: centerjustify-content: flex-endjustify-content: space-betweenjustify-content: space-around

Align-Items

The align-items property specifies the alignment for flex items inside the container, by default stretched to fit.

The baseline value aligns all flex items along the bottom of the font.
